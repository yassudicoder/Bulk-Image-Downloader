FAQ

Questions, answered.

Is it free?
Yes. Every feature is unlocked during the beta, with no paywall and no feature gating. Some power features (like folder rules) are marked "Pro" for the future, but they're open to everyone right now.
Do you see my images?
No. Scanning, de-duplication and ZIP building all happen entirely on your device. Nothing is uploaded to us or anyone else — the extension has no server to send images to.
Which pages does it work on?
Any normal web page. Browser internal pages and the Chrome Web Store are off-limits to every extension, so scanning is disabled there — that's a browser rule, not our choice.
Where do my downloads go?
Straight into your browser's Downloads folder — the same place every other download lands. With folder rules you can route them into tidy subfolders by domain, date or size.
How does duplicate detection work?
Each image gets a tiny perceptual fingerprint (a 64-bit dHash). Images whose fingerprints are close are grouped as near-duplicates, and you choose which copy to keep — usually the largest.
What permissions does it need?
Just the active tab, and only when you click the icon. To fetch images from servers that block cross-origin requests, it can ask for one optional "all sites" permission — but only if some images need it, and only with your say-so.
Privacy

Your images never leave your device.

There's no account to create and no server to phone home to. The extension reads a page only when you click it, does all its work locally, and forgets everything when you close the tab.
