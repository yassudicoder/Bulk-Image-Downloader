FAQ

Questions, answered.

Is it free?+
Yes. Every feature is unlocked during the beta, with no paywall. A few power features are marked "Pro" for later, but they're open to everyone right now.
Do you see my images?+
No. Scanning, de-duplication and ZIP building all happen on your device. There's no server to send images to.
Which pages does it work on?+
Any normal web page. Browser-internal pages and the Chrome Web Store are off-limits to every extension — that's a browser rule, not our choice.
Where do downloads go?+
Into your browser's Downloads folder, like any other download — or tidy subfolders, if you set up folder rules.
How does de-duplication work?+
Each image gets a tiny 64-bit perceptual fingerprint. Images with close fingerprints are grouped, and you keep whichever copy you like — usually the largest.
What permissions does it need?+
Just the active tab, and only when you click the icon. To fetch images from servers that block cross-origin requests, it can ask for one optional grant — only if some images need it, and only with your OK.
Privacy

Your images never leave your device.

There's no account to create and no server to phone home to. The extension reads a page only when you click it, does its work locally, and forgets everything when the tab closes.
